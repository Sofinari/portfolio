Fake Pacman

2023 23/06
This project needs access to the camera to function.

Link doc project GitHub

Link website

The project addresses the interface system for generating augmented reality puppets applied on the hand, with the goal of creating a puppet that extends over the entire hand or part of it, graphically representing a character with its own character.

With the computer camera capturing and recognizing the hands for tracking and drawing the puppet, along with "Media Pipe" (and p5js) technology to create interactive digital puppets.

The developed interface system is designed to be intuitive and easy to use even for inexperienced users, offering a new interactive experience.My project, called Fake-Pacman, is an homage to the original game. However, unlike the original Pacman, the character is not found in a maze, which justifies the attribute "fake".
